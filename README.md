# La Verdadera Historia Del Cine Mexicano

Este proyecto tiene la finalidad de listar las distintas películas, asi como imágenes de los actores que se mencionan durante la serie del cine mexicano en el podcast **La Verdadera Historia de México**.

## Formato JSON para datos

```json
{
  "id": "",
  "title": "",
  "type": "",
  "thumbnail": "",
  "episode": "",
  "time": "",
  "description": ""
}
```
